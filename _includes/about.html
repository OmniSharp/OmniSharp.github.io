About

OmniSharp is a set of tooling, editor integrations and libraries that together create an ecosystem that allows you to have a great programming experience no matter what your editor and operating system of choice may be.

The OmniSharp project has been made by the community for the community, While a few of us work for Microsoft, this project is a community effort and not sponsored or endorsed by Microsoft. The project was initially started by Jason Imison

We say yes!

We unapologetically love .NET. We love its languages and all its libraries and frameworks. We also love Sublime, Atom, Emacs, Vim and Brackets.

We create OWIN middleware on Ubuntu and deploy to Azure. We make ASP.NET services from our Macs and deploy to Linux.

And we do all this while still using useful features like Intellisense, Add Reference, Format Document, and lots more.
